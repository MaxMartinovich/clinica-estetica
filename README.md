## Descrição
Landing Page de uma cliníca ficticia, que apresenta os serviços ofertados por ela, desenvolvida em  **HTML e CSS**.  

## Tecnologias utilizadas
- HTML5
- CSS

## Funcionalidades
- Menu navegação que vai para as partes da página
- Hover nos cards de serviços e profissionais
- Ajuste para todos os tipos de telas(até mobile)
- Interface gráfica amigável

## Como executar o projeto

1ª - Instale alguma IDE de sua escolha, eu utilizei o VSCode

2ª - Copie o código disponibilizado

3ª - Execute a aplicação!
